# Bizerba Scale Management System

A comprehensive system to manage product prices and data for Bizerba weighing scales across multiple store locations.

## Overview

This system manages 8+ Bizerba weighing scales connected via network, allowing centralized product and price management with synchronization capabilities.

## Features

- Multi-scale management
- Product price management
- CSV import/export
- Network synchronization
- Price history tracking
- Web-based interface
